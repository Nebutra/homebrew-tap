# Nebutra Homebrew Tap

Official Homebrew formulae for [Nebutra](https://github.com/Nebutra) projects.

## Carina

Install Carina on macOS:

```bash
brew install Nebutra/tap/carina
```

Upgrade an existing installation:

```bash
brew update
brew upgrade carina
```

Homebrew does not accept a formula name as an argument to `brew update`.
Use `brew update && brew upgrade carina` to refresh Homebrew metadata and
upgrade Carina.

Formula updates are generated from checksummed assets published by the
[Carina release workflow](https://github.com/Nebutra/carina/actions/workflows/release.yml).
